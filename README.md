# ContentHub

ContentHub is open source site bookmarking site that enables you to bookmark all of your websites and articles you want to reference in the future.
You can organize your bookmarks into private or public collections. You can also share your collections, tag them for better searching and triaging.

## Tech Stacks

-   NextJs
-   NextAuth
-   PlanetScale
-   Prisma
-   tRPC
-   tailwind css
-   React query
-   React hooks form
-   Zod
